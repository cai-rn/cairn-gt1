# cairn-gt1
> Repo du Groupe de travail BDD &amp; référentiels du réseau CAI-RN de la MITI (CNRS)


## Liste de diffusion au GT1

### S’inscrire

Pour vous inscrire à la liste de diffusion `cairn-gt1@framagroupes.org`, envoyez un message à `sympa@framagroupes.org` avec cet **objet** (laissez le corps vide ou mettez un court message) :

- SUBSCRIBE cairn-gt1 *Prénom* *Nom*

Remplacez *Prénom* et *Nom* par votre vrai nom ou identifiant, ex : `SUBSCRIBE cairn-gt1@framagroupes.org John Doe`

### Se désinscrire

Pour vous désinscrire de la liste de diffusion `cairn-gt1`, même procédure que ci-dessus, avec cet **objet** :

- UNSUBSCRIBE cairn-gt1
